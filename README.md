# 决策成长日记

一个以“治愈花园”为外壳的个人决策成长工具。它不评判一次选择是“对”还是“错”，而是帮助用户记录决策、推进行动、阶段复盘，并把经验沉淀成可回看的成长片段。

## 当前状态

当前主应用是 Web 版，代码位于 `web-app/`。项目已经完成基础闭环，并进入 V1.2 留存与复盘增强阶段。

已具备的核心能力：

- 决策花园：用种子、发芽、长叶、初开、盛开表达决策成长阶段。
- 决策记录：支持记录背景、选项、选择理由、预期结果和复盘时间。
- 浇水复盘：支持当下复盘、结果复盘、主动复盘和提醒中心。
- 决策锦囊：按不同困境提供思考框架，并可转成决策记录。
- 决策风格测试：包含快速版、标准版、深入版题库。
- 成长片段：从复盘中沉淀可回看的经验。
- 本地保存：使用浏览器本地存储，并支持 JSON 导入导出。

## 产品原则

- 保留种花和治愈感，但花园只是外壳。
- 核心价值来自决策记录、行动推进、复盘和自我洞察。
- 不用“成功/失败”“选对/选错”评价用户。
- 花朵成长绑定记录、行动、复盘和学习，而不是绑定最终结果好坏。
