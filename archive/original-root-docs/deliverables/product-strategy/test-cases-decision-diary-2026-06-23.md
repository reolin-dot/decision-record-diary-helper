## 1. 测试范围与策略

### 1.1 功能测试范围

| 测试范围 | 包含内容 | 优先级 |
|---------|---------|--------|
| **花园页** | 决策展示、种花/浇水/锦囊按钮、最近决策列表 | P0 |
| **记录决策** | 4步向导（标题→选项→选择→复盘时间） | P0 |
| **复盘功能** | 复盘提醒、复盘填写、跳过复盘 | P0 |
| **锦囊** | 困境选择、分析框架、结果保存 | P1 |
| **个人中心** | 数据统计、成就徽章、决策风格 | P1 |
| **微信登录** | 授权登录、用户信息获取 | P0 |
| **数据持久化** | 决策记录存储、复盘记录存储 | P0 |

### 1.2 兼容性测试范围

| 测试类型 | 测试范围 | 优先级 |
|---------|---------|--------|
| **iOS 兼容性** | iPhone X 及以上机型，iOS 14+ | P0 |
| **Android 兼容性** | 主流 Android 机型（华为、小米、OPPO、vivo） | P0 |
| **微信版本** | 微信 8.0+ | P0 |
| **屏幕尺寸** | 小屏（375px）、中屏（414px）、大屏（428px+） | P1 |
| **深色模式** | iOS/Android 深色模式适配 | P2 |

### 1.3 性能测试范围

| 测试指标 | 目标值 | 优先级 |
|---------|--------|--------|
| **启动时间** | < 2 秒（冷启动） | P0 |
| **页面加载时间** | < 1 秒 | P0 |
| **内存占用** | < 200MB | P1 |
| **API 响应时间** | < 500ms（P95） | P0 |

### 1.4 异常测试范围

| 测试类型 | 测试场景 | 优先级 |
|---------|---------|--------|
| **网络异常** | 断网、弱网、超时 | P0 |
| **数据异常** | 后端返回异常数据、数据缺失 | P1 |
| **用户操作异常** | 快速点击、中途退出、横竖屏切换 | P1 |
| **微信环境异常** | 微信登录失败、订阅消息失败 | P1 |

### 1.5 测试策略

| 测试阶段 | 时间 | 内容 |
|---------|------|------|
| **单元测试** | 开发阶段 | 关键工具函数、数据模型 |
| **集成测试** | 开发完成后 | API 接口、前后端联调 |
| **系统测试** | 提测后 | 功能测试、异常测试、兼容性测试 |
| **回归测试** | 每次发版前 | 核心流程回归 |
| **灰度测试** | 发布前 | 小范围用户真实环境测试 |
