# Tab Bar 图标说明

需要准备 6 个 tab 图标文件（81x81px PNG 格式）：

- `tab-garden.png` - 花园图标（未选中状态，灰色）
- `tab-garden-active.png` - 花园图标（选中状态，黑色）
- `tab-coach.png` - 锦囊图标（未选中状态，灰色）
- `tab-coach-active.png` - 锦囊图标（选中状态，黑色）
- `tab-profile.png` - 我的图标（未选中状态，灰色）
- `tab-profile-active.png` - 我的图标（选中状态，黑色）

建议图标风格：
- 线条风格，2px 线宽
- 未选中：#888888
- 选中：#222222
- 花园可用 🌿 叶子造型
- 锦囊可用 📦 盒子造型
- 我的可用 👤 人形造型
