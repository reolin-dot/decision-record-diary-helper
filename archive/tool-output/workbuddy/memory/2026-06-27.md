#### 1. 新建决策列表页面 `pages/decision-list/`

- 在 `app.json` 注册页面

#### 2. 修改个人中心 `pages/profile/`
- **profile.js**:
  - 去掉设置列表里所有 `disabled: true`，启用"决策记录"入口
  - 新增 `loadStats()` 方法，正确计算统计数据：
    - `reviewRate`: 已复盘数 / 总决策数
    - `growthLoopRate`: 已复盘中决策正确的比例（rating 为"超出预期"或"符合预期"）
    - `streak`: 简化计算，暂时写死为 1
  - `onTapSetting()` 支持跳转到决策列表页面
- **profile.wxml**:
  - 去掉数据统计区域的 `wx:if`（即使没有数据也显示 0）
  - 去掉设置列表的 `disabled` 相关逻辑

#### 3. 数据统计逻辑说明
- **复盘完成率** = 已复盘决策数 / 总决策数
- **决策正确率** = 已复盘中结果符合或超出预期的比例
- **连续天数** = 暂时简化，后续可以改成真实计算
