# 决策成长日记 Web App 更新日志

## V2.0-alpha.8 主题下一步建议 - 2026-07-07

### 新增

- 主题花园新增“下一步建议”，根据当前主题下的决策状态给出行动入口。
- 有到期复盘时优先引导到提醒中心；有收获后引导继续记录。
- 新增主题下一步建议领域逻辑和测试。

### 验收重点

- 无主题记录、到期复盘、已有主题收获三种场景都有明确下一步。
- `npm test` 和 `npm run build` 通过。

## V2.0-alpha.7 主题收获沉淀 - 2026-07-06

### 新增

- 主题花园新增“主题收获”，展示该主题下最近沉淀的成长片段。
- 新增按主题筛选成长片段的领域逻辑和测试。

### 验收重点

- 主题收获只展示当前主题下的复盘片段。
- 没有片段时展示明确空状态。
- `npm test` 和 `npm run build` 通过。

## V2.0-alpha.6 主题洞察提示 - 2026-07-06

### 新增

- 主题花园新增“主题洞察”文案，根据主题数量和占比给出一句温和观察。
- 新增主题洞察领域逻辑和测试，避免页面里堆判断。

### 验收重点

- 高占比主题、新出现主题和空主题都有可读提示。
- `npm test` 和 `npm run build` 通过。

## V2.0-alpha.5 主题花园雏形 - 2026-07-06

### 新增

- 新增主题花园页面，按主题查看对应决策列表。
- 成长档案页的主题统计支持点击进入对应主题。
- 月度报告、成长档案、主题统计开始形成“长期成长档案”的基础结构。

### 验收重点

- 主题花园能展示全部已有主题，并默认选中数量最多的主题。
- 点击主题后，只展示该主题下的正式决策。
- `npm test` 和 `npm run build` 通过。

## V2.0-alpha.4 成长档案主题统计 - 2026-07-06

### 新增

- 成长档案页新增“主题统计”，展示用户最常记录的决策主题。
- 主题统计按决策数量排序，并显示占比进度条。
- 新增主题统计领域逻辑和测试，为后续主题花园铺路。

### 验收重点

- 草稿和软删除记录不进入主题统计。
- 锦囊生成和自由记录都有可读主题标签。
- `npm test` 和 `npm run build` 通过。

## V2.0-alpha.3 成长档案历史索引 - 2026-07-06

### 新增

- 成长档案页新增“历史月报”列表，直接展示最近可回看的月份。
- 点击历史月报可进入对应月份的月度成长报告。
- 月度成长报告支持通过 `month` 查询参数打开指定月份。

### 验收重点

- 成长档案能作为月报历史索引使用。
- 从历史月报进入后，刷新页面仍停留在对应月份。
- `npm test` 和 `npm run build` 通过。

## V2.0-alpha.2 月度报告历史回看 - 2026-07-06

### 新增

- 月度成长报告新增月份切换，支持查看当前月、上个月以及有记录或复盘的历史月份。
- 月报统计、主题和成长片段会跟随所选月份重新计算。

### 验收重点

- 有跨月决策和复盘时，月份列表按新到旧展示。
- 切换历史月份后，月报只统计该月内容。
- `npm test` 和 `npm run build` 通过。

## V2.0-alpha.1 月度成长系统起点 - 2026-07-06

### 新增

- 新增月度成长报告页，汇总本月决策数、复盘次数、成长片段、本月主题和下月先做事项。
- 新增成长档案入口页，聚合月度报告、成长片段、AI 洞察和决策记录。
- 个人页新增“成长档案”和“月度成长报告”入口。

### 说明

- 这是 2.0 的第一块能力，不包含云同步、账号体系、分享卡片或完整主题花园。

### 验收重点

- 月报在无数据、有本月记录、有本月复盘片段时都有可读内容。
- 成长档案能跳转到月报、成长片段、AI 洞察和决策记录。
- `npm test` 和 `npm run build` 通过。

## V1.8 复盘完成反馈版 - 2026-07-06

### 新增

- 新增复盘成功页，完成浇水后展示本次复盘类型、决策标题、本次沉淀和状态。
- 如果还有后续跟进复盘，成功页展示剩余次数和下次提醒日。
- 成功页新增“查看成长片段”入口，让复盘后的经验能被马上找回。

### 验收重点

- 完成当下复盘或结果复盘后，不再直接回首页，而是进入复盘成功页。
- 无收获文本时有兜底文案，不出现空白成功页。
- `npm test` 和 `npm run build` 通过。

## V1.7 留存与复盘体验增强版 - 2026-07-06

### 新增

- 首页新增“今天最值得回看”卡片，优先展示最需要复盘、行动跟进或结果回看的决策。
- 成长片段页新增“全部 / 当下复盘 / 结果复盘”筛选。
- 成长片段页新增关键词搜索，支持按片段正文、摘要、来源决策和复盘状态查找。

### 优化

- 将 Web 应用版本号更新为 `1.7.0`。
- 保留 V1.6 的本地数据安全能力：数据健康检查、快速备份、完整备份、导入预览、低风险修复、DeepSeek 分析包和 AI 洞察回填。

### 验收重点

- 首页有待回看的决策时，能直接进入对应复盘页。
- 成长片段筛选和搜索组合使用时，结果准确。
- `npm test` 和 `npm run build` 通过。

## V1.6 数据安全与本地闭环增强版 - 2026-07-06

- 新增本地数据健康检查、备份新鲜度提示、快速备份和完整本地备份。
- 新增导入前影响预览，导入或修复前自动下载当前备份。
- 新增低风险数据修复，只处理空数组、默认状态、布尔字段等格式问题。
- 新增 DeepSeek 分析包导出和 AI 成长洞察保存。
- 新增个性化复盘问题、复盘状态选项、跟进提示和首页决策模式洞察。
