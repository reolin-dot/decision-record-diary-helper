决策成长日记 Web App V1.6 点击流原型

基于当前 React Web V1.6：花园首页、记录决策、锦囊、提醒复盘、成长片段、个人页、数据安全、DeepSeek 分析与 AI 洞察回填。

当前定位：V1.6 数据安全与本地闭环增强版。核心闭环是“记录决策 -> 开始行动 -> 个性化复盘 -> 沉淀成长片段 -> 本地备份/恢复 -> 外部 AI 洞察回填”。
首页 种花记录 详情行动 提醒复盘 锦囊分析 成长资产 个人与数据
点击关系总览
点击“种花”新用户先进入决策风格测试；已完成或跳过测试后进入 4 步记录页。
点击“提醒”进入提醒中心，按到期复盘、行动提醒、结果跟进分组展示，并带入个性化复盘问题。
点击“锦囊”进入困境选择页，选中困境后填写分析，再生成锦囊卡片。
点击花朵/最近决策进入决策详情，可标记开始行动、继续草稿或进入复盘。
完成复盘弹出跟进次数选择，保存后返回花园，经验进入成长片段。
个人页设置入口通向提醒中心、成长片段、数据导出、风格测试、决策记录。
点击“数据导出与 AI 分析”进入 V1.6 数据安全页，可做健康检查、快速备份、完整备份、导入预览、低风险修复。
点击“复制并打开”生成 DeepSeek 可读分析包，复制后打开外部 AI，再把分析结果保存为成长洞察。
V1.6 版本新增范围
本地数据安全新增数据健康检查、备份新鲜度提示、快速备份、完整备份导出。
导入与修复导入前预览新增/合并影响；导入或修复前自动下载当前备份；低风险格式问题可自动修复。
AI 辅助闭环导出 DeepSeek 分析包，保存 AI 返回的成长洞察，并在首页和个人页露出。
个性化复盘根据决策风格和历史记录调整复盘问题、状态选项和跟进提示。
决策模式洞察首页展示近期决策模式卡片，引导用户回到锦囊、记录或数据分析。
仍不做的事不强制登录，不做真实云同步，不在产品内直接替用户调用 AI 做最终决策。
